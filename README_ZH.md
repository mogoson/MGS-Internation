# MGS-Multilingualism
- [English Manual](./README.md)

## 概述
- 多语言文本段落工具。

## 环境
- Unity 5.0 或更高版本。
- .Net Framework 3.5 或更高版本。

## 实现
- 本地文本文件存储多语言文本段落。
- 通过预定义Key获取当前设置语言下的文本段落。

## 案例
- "MGS-Multilingualism/Scenes" 目录下存有上述功能的演示案例，供读者参考。
